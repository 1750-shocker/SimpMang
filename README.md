# SimpMang

A manga/comic reading Android app with OCR text recognition and Chinese text conversion capabilities.

## Features

- **Baidu OCR SDK** — Recognizes text from manga/comic images using Baidu's Optical Character Recognition service
- **OpenCC** — Converts recognized Chinese text between Simplified and Traditional Chinese

## Tech Stack

- Jetpack Compose + Material 3
- Baidu OCR SDK
- OpenCC (Chinese text conversion)
